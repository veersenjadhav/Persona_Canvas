# **App Name**: Persona Canvas

## Core Features:

- About Me: A landing page introducing the user with a brief professional summary.
- Academic Records: Display educational qualifications, institutions attended, and relevant coursework.
- Professional Timeline: Showcase previous employment, roles held, and responsibilities.
- Projects and Contributions: List personal projects with descriptions, technologies used, and links to repositories or live demos.
- Recognition Showcase: Display certifications, awards, and other recognitions with descriptions and issue dates.
- Photographic Gallery: Present best photographs in a visually appealing grid or slideshow. Use generative AI tool to create captions based on image recognition.
- Get In Touch: A contact form and display of the user's contact details, including email, phone number, and social media links.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3A59) to convey professionalism and trustworthiness.
- Background color: Very light blue-gray (#F0F4F8) for a clean, modern feel.
- Accent color: Muted teal (#45A29E) for highlights and interactive elements, offering a touch of creativity.
- Headline font: 'Space Grotesk', sans-serif, for a modern, technical feel.
- Body font: 'Inter', sans-serif, for a clean, readable text.
- Use a set of minimalist, line-based icons to represent different sections and skills.
- Employ a grid-based layout to maintain structure and balance. Use white space generously to create a sense of clarity.
- Incorporate subtle scroll animations to create an engaging and dynamic user experience. Elements can fade in, slide, or scale as the user progresses down the page.
